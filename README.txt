1. Клонировать репозиторий GitHub https://github.com/Krotijkee/test_project.git
2. Запустить проект в среде разработки IntelliJ IDEA
3. Выделить класс SignUpFormTests правой кнопкой мыши
4. В раскрывшемся меню выбрать 'Run LoginFormTest'
5. После того, как прохождение всех тестов будет завершено - запустить в maven плагин allure:install
6. После выполнения allure:install запустить allure:report
7. После того как allure:report выполнится, чтобы просмотреть allure отчёт необходимо развернуть веб-сервер. Для этого запустить утилиту allure:serve
